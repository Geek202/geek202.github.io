A rust program that wraps a Minecraft server and automatically restarts it if it crashes.

When my Minecraft server kept crashing (tip: Raspberry Pi is not the best for running one), I decided to create a wrapper that could automatically restart the server for me. The concept originally included support for Discord webhooks and a small mod that adds a command to safely restart the server, but I then expanded the mod to add support for things such as restarting based on a cron schedule, and adding a command to restart when there is no players online.

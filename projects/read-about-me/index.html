A website for creating GitHub profile README cards

read-about-me is a simple website created using next.js to generate nice cards to display on your GitHub profile README, or anywhere else you can embed an image, such as like this:

You can create your own here, and it will automatically generate both a direct link to an SVG that you can embed, a link you can share that displays it in a webpage, and even a markdown snippet to embed the image!

You can check out my card in action on my GitHub profile here!
